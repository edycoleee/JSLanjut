Penggunaan Display

@edy cole emailku tanggal jam

setiap tag pada HTML berada dalam sebuah kotak. properti display pada CSS mengatur perilaku dari kotak tersebut. value display ada4: inline, inline block, block, none

inline -secara default tidak menambahkan baris baru -lebar dan tinggi elemennya sebesar konten yang ada di dalamnya -kita tidak dapat mengatur lebar dan tinggi elemen inline -margin dan padding hanya mempengaruhi elemen secara horizontal, tidak vertikal -elemen default: b,strong,i ,em,a,span,button,input,label,select,textarea lihat contoh inline => a tidak menambah baris -hanya satu elemen => img adalah inline yang bisa di setting lebar dan tinggi

inline-block -tidak ada elemen yg secara default memiliki inline-block -kita harus ubah secara manual properti tsb -perilaku dasarnya sama dengan elemen inline -perbedaannya, elemen inline-block dapat kita atur tinggi dan lebar-nya contohnya pada a biasanya utk navigasi mengubah dari inline menjadi block

block -element HTML yang secara default menambahkan baris baru ketika dibuat -jika tidak diatur lebarnya, maka lebar default element block akan memnuhi lebar dai browser / parent nya -kita dapat megatur tinggi dan lebar elemen block -di dalam elemen block, kita dapat menyimpan tag dengan elemen inline, inline-block, atau menambahkan elemen block lagi -contohnya h1-h6, p, ol, ul, li,form, hr, div
